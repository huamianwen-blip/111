# 51单片机智能温控风扇系统

## 项目简介
本项目基于STC89C51/C52单片机，集成DS18B20温度传感器和PWM调速控制，实现一个智能温控风扇系统。

## 主要功能
- ✅ DS18B20温度采集
- ✅ PWM调速控制（0-5档）
- ✅ 自动温控模式（温度≥28°C自动启动）
- ✅ 按键手动调节风扇档位
- ✅ LCD1602显示温度和状态
- ✅ 定时功能
- ✅ 报警功能

## 硬件配置
- 单片机：STC89C51/C52
- 温度传感器：DS18B20
- 液晶显示：LCD1602
- 风扇控制：PWM或继电器
- 按键：4个独立按键

## 文件结构
```
├── main.c              # 主程序
├── ds18b20.c/h         # DS18B20温度传感器驱动
├── pwm.c/h             # PWM调速控制
├── lcd1602.c/h         # LCD1602液晶显示驱动
├── key.c/h             # 按键控制
├── delay.c/h           # 延时函数
└── config.h            # 系统配置文件
```

## 快速开始
1. 用Keil uVision 5打开项目
2. 编译代码
3. 用Proteus仿真或烧录到单片机

## 注意事项
- 根据实际硬件调整引脚定义
- 修改config.h中的温度阈值
- 确保晶振频率设置正确
