# DS18B20温度传感器详解

## 基本特性
- 工作温度范围: -55°C ~ +125°C
- 精度: ±0.5°C (-10°C ~ +85°C)
- 分辨率: 9/10/11/12位可选（默认12位）
- 通信: 单总线接口
- 供电: 3V ~ 5.5V

## 单总线时序

### 初始化时序
```
主机: 发送复位脉冲 (480us低电平)
DS18B20: 返回应答脉冲 (60-240us低电平)
```

### 读写时序
```
写0: 60us低电平 + 最少1us高电平
写1: 最少1us低电平 + 最少68us高电平

读0: 15us低电平 + 45us读取
读1: 1us低电平 + 读取高电平
```

## 主要命令
- 0x33: READ ROM - 读取ROM
- 0xCC: SKIP ROM - 跳过ROM
- 0x44: CONVERT T - 启动温度转换
- 0xBE: READ SCRATCHPAD - 读取温度数据
- 0x4E: WRITE SCRATCHPAD - 写入温度阈值

## 温度读取流程
1. 复位DS18B20
2. 发送SKIP ROM命令(0xCC)
3. 发送CONVERT T命令(0x44)
4. 等待转换完成(最多750ms)
5. 复位DS18B20
6. 发送SKIP ROM命令(0xCC)
7. 发送READ SCRATCHPAD命令(0xBE)
8. 读取两字节温度数据
9. 温度 = (温度高字节 + 温度低字节/256) * 0.0625

## 代码示例
```c
float temp = ds18b20_read_temp();  // 返回温度值，单位°C
```
